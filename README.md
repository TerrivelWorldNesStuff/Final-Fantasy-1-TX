# Final-Fantasy-1-TX
A list of small projects, patches and hacks for one of JRPG grandfathers.

INTRODUCTION
At certain point of 2026, I decided to play EVERY possible rom hack available for Final Fantasy 1, from the Nintendo Entertainment System. I was sobber, which is worse.

To resume what already is a colossal story: This is the depository of many files related to that journey, from personal findings, hacks lost in time or even files that I have to do it myself. Each folder holds a type of file and the README on those folder gives further details on what each file does.
